## 📖 Descripció del projecte

**M.S.i.X Hub** és una plataforma web progressiva (PWA) que integra múltiples aplicacions de suport per a la **gestió forestal de camp i despatx**. Està dissenyada per a professionals del sector (enginyers de boscos, tècnics forestals, estudiants i investigadors) que necessiten eines digitals per recollir, analitzar i simular dades forestals de manera eficient.

El projecte neix de la necessitat de **digitalitzar els processos tradicionals d'inventari forestal** i oferir una experiència unificada que combini recollida de dades al camp, anàlisi estadística i simulació de models forestals.

## 🛠️ Aplicacions incloses

### 📱 Aplicacions de camp

| App | Descripció | Versió |
|-----|-----------|--------|
| 🌲 **Inventari Forestal** | Recollida de dades per parcel·les: dasometria, vegetació, topografia, sòl i observacions. Exportació a Excel. | v10 |
| 🎯 **Relascopi de Bitterlich** | Càlcul d'àrea basal mitjançant relascopi virtual amb càmera i sensor d'inclinació. | v1 |
| ⚡ **Alçades d'arbres** | Mesura d'alçades amb inclinòmetre digital i càmera. | v1 |
| 📋 **Pre-Inventari** | Planificació d'inventaris: generació de punts de mostreig (quadrícula/aleatori) a partir de GeoJSON de rodals. | v1 |

### 🖥️ Aplicacions de despatx

| App | Descripció | Versió |
|-----|-----------|--------|
| 🌳 **SIMBOSCMED** | Simulador de boscos mediterranis (Shiny/medfate). | Shiny |
| 📥 **Inputs** | Generador de fitxers d'entrada per a models de simulació (Shiny). | Shiny |
| 📊 **Eina d'Anàlisi** | Anàlisi estadística de dades d'inventari forestal (Shiny). | v4 |
| 🧠 **SylvaMind AI** | Assistent intel·ligent especialitzat en gestió forestal mediterrània. | IA |

## ✨ Funcionalitats principals

- ✅ **Funcionament 100% offline** — Recull dades al camp sense connexió a internet.
- ☁️ **Sincronització al núvol** — Dades segures amb Supabase (per a usuaris registrats).
- 🌍 **Multilingüe** — Disponible en català, castellà i anglès.
- 📤 **Exportació de dades** — Formats Excel (XLSX), CSV i GeoJSON.
- 📍 **Geolocalització GPS** — Coordenades integrades a les mesures.
- 📱 **PWA (Progressive Web App)** — Instal·lable al mòbil com una app nativa.
- 🔒 **Privacitat garantida** — Cada usuari només pot veure les seves pròpies dades.

## 🚀 Com utilitzar l'aplicació

### 1. Accés via web (recomanat)

1. Obre el navegador del teu mòbil o ordinador.
2. Ves a: [https://msix2003.github.io/prova-de-apps/](https://msix2003.github.io/prova-de-apps/)
3. Inicia sessió amb un compte o continua com a convidat.
4. Selecciona l'aplicació que necessitis des del Hub.

### 2. Instal·lació com a app (PWA)

**Android (Chrome):**
1. Obre la web al navegador Chrome.
2. Toca el menú (⋮) → "Afegeix a la pantalla d'inici".
3. L'app s'instal·larà amb icona i funcionament natiu.

**iOS (Safari):**
1. Obre la web a Safari.
2. Toca el botó de compartir (□↑) → "Afegeix a la pantalla d'inici".
3. Confirma amb "Afegeix".

### 3. Ús offline

- Les aplicacions de camp (Inventari, Relascopi, Alçades) funcionen **sense connexió**.
- Les dades es guarden al navegador (`localStorage`).
- Quan recuperis connexió, les dades es sincronitzaran automàticament (només usuaris registrats).

### 4. Exportació de dades

- Des de l'app d'**Inventari Forestal**, prem "Descarregar Excel" al panell "Desar".
- Els fitxers es descarreguen directament al teu dispositiu.

## 🏗️ Arquitectura tècnica

```
prova-de-apps/
├── index.html          # Hub principal (login + navegació)
├── manifest.json       # Configuració PWA
├── sw.js               # Service Worker (offline)
├── Inventari.html      # App d'inventari forestal
├── Relascopi.html      # Relascopi de Bitterlich
├── altimetre.html      # Mesura d'alçades
├── Pre_Inv.html        # Planificador d'inventaris
├── simboscmed.html     # Wrapper SIMBOSCMED (Shiny)
├── inputs.html         # Wrapper Inputs (Shiny)
├── analisi.html        # Wrapper Eina d'Anàlisi (Shiny)
├── sylvamind.html      # Assistent IA SylvaMind
├── peticions.html      # Formulari de propostes/millores
├── dubtes.html         # FAQ interactiva (ForestBot)
├── favicon.png         # Icona favicon
├── icon-*.png          # Icones PWA (180, 192, 512)
└── README.md           # Aquest fitxer
```

### Tecnologies utilitzades

- **Frontend:** HTML5, CSS3, JavaScript (vanilla)
- **Backend/BBDD:** Supabase (PostgreSQL + Auth)
- **Models:** Shiny (R) per a simuladors
- **PWA:** Service Worker, Web App Manifest
- **CDN:** Google Fonts, SheetJS (Excel), Supabase JS Client
